# 개인 사이드 프로젝트

## 📝 [Plan of P] 계획 작성 웹,앱

## ❓구현 내용

#### React, Bootstrap 활용하여 반응형 구현

#### 카카오 로그인, OAuth 로그인 구현

#### 더치페이 기능 구현

#### 카카오 맵 키워드 검색, 주소 검색을 통해 길찾기 기능 구현

#### 캘린더 기능 구현, 작성한 계획,더치페이를 달력을 통해 확인가능

#### Node, Express, MongoDB를 사용하여 서버 API 코드를 구현

#### AWS EC2, Ngnix 를 사용하여 서비스 배포
